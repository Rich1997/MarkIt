# **App Name**: Reactify

## Core Features:

- UI Components: Provides a library of reusable and customizable UI components.
- Theming Support: Allows users to easily switch between light and dark themes.
- Responsive Design: Ensures the application is accessible and usable on different screen sizes and devices.

## Style Guidelines:

- Dark color scheme.
- Primary color: Royal Blue (#4169E1) for a sophisticated feel.
- Background color: Dark gray (#282828) for a modern, tech-focused appearance.
- Accent color: Teal (#008080) to provide a contrasting highlight.
- Font: 'Inter', a sans-serif, for both headlines and body text.
- Use simple and consistent icons.
- Use subtle animations to create a polished experience.
